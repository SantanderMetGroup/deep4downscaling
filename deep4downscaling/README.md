# deep4downscaling package

This folder contains the Python source code of the `deep4downscaling` library.

## Main modules

- `trans.py`: Data preprocessing and transformation utilities.
- `viz.py`: Visualization helpers for downscaling workflows.
- `metrics.py` and `metrics_ccs.py`: Evaluation metrics for historical performance and climate change signal analysis.
- `deep/`: Deep learning utilities, including training, prediction, model definitions, losses, and XAI tools.

The library is structured so that common downscaling operations can be reused across different experiments and model architectures.
